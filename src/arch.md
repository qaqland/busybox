# arch

## SYNOPSIS

    arch

## DESCRIPTION

输出系统的架构。

不同架构运行不同的系统，系统中运行软件所需要的安装包也不同。
如 [_Alpine Linux downloads_](https://alpinelinux.org/downloads/)
有为不同架构提供的系统镜像，个人电脑常见的架构有：

* x86_64 / amd64
* aarch64 / arm64

## EXAMPLES

    $ arch
    x86_64
